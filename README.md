# iPhone Website Project

This is a responsive **iPhone-themed website** built using **HTML, CSS, and JavaScript**. The website showcases the layout, design, and features of an iPhone landing page, including sections like product display, features, specifications, and footer. It is designed to be **fully responsive**, providing a smooth user experience across desktop and mobile devices.

## Features
- Modern, clean design inspired by iPhone landing pages
- Fully responsive layout for all screen sizes
- Interactive elements using CSS and JavaScript
- Smooth navigation between sections

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Purpose
This project demonstrates front-end development skills, including:
- Responsive design
- Layout structuring
- Interactive UI implementation
